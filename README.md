# Android TV APK menu

Public menu:

https://tyranosaur.github.io/tv-apk-menu/

## Add an APK

1. Copy the APK file into `apks`.
2. Run `.\publish.ps1`.
3. Wait about a minute for GitHub Pages to update.

## Remove an APK

1. Delete the APK file from `apks`.
2. Run `.\publish.ps1`.
3. Wait about a minute for GitHub Pages to update.
